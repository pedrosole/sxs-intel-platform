## Escopo

### IN
- Criar funcao `parseCalendarOutput(raphaOutput: string, jobId: string, clientId: string, monthYear: string)` que:
  1. Parseia o Markdown do @rapha extraindo cada peca (titulo, formato, cluster, subtema, objetivo, dia/semana)
  2. Mapeia formatos para os valores validos: post, story, carrossel, reel, blog, linkedin, especial
  3. Mapeia objetivo para: atrair, educar, converter, fidelizar

  - Emitir evento com o link: `{ type: "calendar_link", url: "/calendario/{token}" }`
- No `ChatArea.tsx`, tratar evento `calendar_link` e exibir botao/link clicavel

### OUT
