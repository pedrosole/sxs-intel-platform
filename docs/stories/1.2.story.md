# Story 1.2 — Calendar Persistence Bridge (Rapha → DB → Link)

**Status:** Ready
**Tipo:** Feature
**Prioridade:** CRITICAL
**Estimativa:** M (media)

---

## Descricao

Quando o pipeline roda, @rapha produz o calendario editorial como texto Markdown no chat. Esse output NAO e parseado e NAO e salvo nas tabelas `calendar_pieces` e `calendar_meta`. O link publico do calendario (`/calendario/{token}`) nunca e gerado.

**Evidencia:** Apos 2 execucoes completas da Gabriela Lopes, `calendar_meta` e `calendar_pieces` estao VAZIOS no Supabase.

O fluxo correto deve ser:
```
@rapha output → parser extrai pecas → salva calendar_pieces + calendar_meta → gera share_token → Hermes entrega link no chat
```

---

## Escopo

### IN
- Criar funcao `parseCalendarOutput(raphaOutput: string, jobId: string, clientId: string, monthYear: string)` que:
  1. Parseia o Markdown do @rapha extraindo cada peca (titulo, formato, cluster, subtema, objetivo, dia/semana)
  2. Mapeia formatos para os valores validos: post, story, carrossel, reel, blog, linkedin, especial
  3. Mapeia objetivo para: atrair, educar, converter, fidelizar
  4. Atribui `day` (1-31) baseado na semana/dia da semana
  5. Atribui `sort_order` dentro do mesmo dia
- Criar funcao `saveCalendarToDb(pieces, jobId, clientId, monthYear)` que:
  1. Cria registro em `calendar_meta` (job_id, client_id, month_year) — share_token e auto-gerado pelo DB
  2. Insere todas as pecas em `calendar_pieces`
  3. Retorna o `share_token`
- Integrar no `orchestrator.ts`:
  - Apos @rapha completar no pipeline, chamar o parser + save
  - Emitir evento com o link: `{ type: "calendar_link", url: "/calendario/{token}" }`
- No `ChatArea.tsx`, tratar evento `calendar_link` e exibir botao/link clicavel

### OUT
- Mudancas na pagina `/calendario/[token]` (ja funciona se tiver dados)
- Mudancas no schema SQL (tabelas ja existem)
- Parser de outputs de outros agentes

## Detalhes Tecnicos

### Formato tipico do output @rapha (para referencia do parser)

```markdown
### **SEMANA 1: TITULO**
**Segunda-feira**
- **Cluster:** Nome do Cluster
- **Subtema:** Subtema especifico
- **Formato:** Carrossel (5 slides) | Reel | Estatico
- **Titulo:** "Titulo da peca"
- **Objetivo:** Atrair | Educar | Converter
```

### Mapeamento de formatos
| @rapha output | calendar_pieces.format |
|---------------|----------------------|
| Carrossel | carrossel |
| Reel | reel |
| Estatico / Post | post |
| Stories | story |
| Blog | blog |
| LinkedIn | linkedin |

### Mapeamento de dias (semana → day)
- Semana 1: seg=dia 1 do mes, qua=3, sex=5
- Semana 2: seg=8, qua=10, sex=12
- Semana 3: seg=15, qua=17, sex=19
- Semana 4: seg=22, qua=24, sex=26

### Novo evento SSE
```typescript
{ type: "calendar_link", token: string, url: string, totalPieces: number }
```
