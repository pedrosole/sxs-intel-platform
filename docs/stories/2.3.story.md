# Story 2.3 — Asset Management UI (4 Abas)

**Epic:** 2 — Design Studio
**Tipo:** Feature
**Prioridade:** CRITICAL
**Estimativa:** G (grande)
**Depende de:** 2.2
**Status:** Draft

---

## Descrição

Criar interface de gestão de assets na ficha do cliente com 4 abas: Logos, Cores, Fontes e Referências. Acessível via `/clientes/[slug]/assets`.

## Acceptance Criteria

### Navegação
- [ ] Página `/clientes/[slug]/assets` criada
- [ ] Link na página `/clientes` para acessar assets de cada cliente
- [ ] Tabs component com 4 abas: Logos | Cores | Fontes | Referências

### Tab 1 — Logos
- [ ] Drag & drop ou botão para upload de logo (PNG/SVG)
- [ ] Campo para atribuir variante: horizontal, vertical, selo, mono
- [ ] Preview das logos uploadadas (thumbnail grid)
- [ ] Botão "Preparar Assets" — chama API prepare, mostra progresso
- [ ] Após preparo: mostra grid com original / cropped / white lado a lado
- [ ] Delete individual

### Tab 2 — Cores
- [ ] Color picker para cor primária
- [ ] Botão "+" para adicionar cores secundárias com label (cream, gold, etc.)
- [ ] Preview da paleta completa (swatches)
- [ ] Auto-derivação dos 6 tokens (BRAND_PRIMARY → LIGHT_BG etc.)
- [ ] Preview dos tokens derivados
- [ ] Salvar como `colors.json` via API

### Tab 3 — Fontes
- [ ] Upload de arquivos .ttf / .otf / .woff2
- [ ] Dropdown para atribuir papel: Heading / Body / Accent
- [ ] Preview tipográfico ao vivo: "Aa Bb Cc 123 — Gabriela Lopes Advocacia"
  - Renderiza com @font-face da fonte uploadada
- [ ] Se não há fonte: sugestão automática de Google Fonts por tom do brand
- [ ] Badge visual: "Fonte própria" vs "Google Fonts (aproximação)"
- [ ] Delete individual

### Tab 4 — Referências
- [ ] Upload múltiplo de imagens (drag & drop batch)
- [ ] Campo de notas/tags por referência
- [ ] Grid visual (masonry ou 3 colunas)
- [ ] Lightbox para ver em tamanho maior
- [ ] Delete individual

### Geral
- [ ] Loading states em todos os uploads
- [ ] Toast de confirmação/erro
- [ ] Consistente com design system existente (glass cards, dark theme)

## Scope

**IN:** Página, 4 abas, upload, preview, delete, prepare-assets
**OUT:** Integração com Design Studio (story 2.5), edição avançada de imagens

## Notas Técnicas

- Usar `shadcn/ui` Tabs component
- Preview de fonte via `@font-face` injetado em `<style>` com URL do Storage
- Color picker: pode usar input[type=color] + hex text input
- Drag & drop: usar `onDrop` nativo + `e.dataTransfer.files`
