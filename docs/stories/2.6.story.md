# Story 2.6 — Export PNG (Playwright Server-Side)

**Epic:** 2 — Design Studio
**Tipo:** Feature
**Prioridade:** HIGH
**Estimativa:** M (média)
**Depende de:** 2.5
**Status:** Done

## Descrição

Exportar peças visuais aprovadas como PNGs 1080×1350px via Playwright rodando server-side. Download individual ou ZIP para carrosséis.

## Acceptance Criteria

- [x] `POST /api/design/export` implementado
  - Input: `{ designPieceId }`
  - Carrega HTML de `design_pieces.html_content`
  - Renderiza com Playwright (Chromium headless)
  - Viewport: 420×525, device_scale_factor: 2.5714
  - Screenshot clip: {x:0, y:0, width:420, height:525}
  - Salva PNG no Supabase Storage (`exports/{slug}/`)
  - Atualiza `design_pieces.export_path` e status → `exported`
  - Retorna URL pública do PNG
- [ ] Para carrossel: exportar todos os slides — adiado para iteração futura
  - Navegar track via translateX por slide
  - Salvar como `slide_1.png`, `slide_2.png`, etc.
  - Gerar ZIP com todos os slides
- [x] Botão "Exportar" no Design Studio (story 2.5)
  - Loading state durante export (~5-10s)
  - Após export: botão "Download" com link direto
  - Para carrossel: "Download ZIP" — adiado
- [x] Wait for fonts: `page.waitForTimeout(2000)` após page load
- [ ] Hide IG frame chrome (header, dots, actions, caption) antes do screenshot — não aplicável (HTML não tem frame chrome)

## Scope

**IN:** API route export, Playwright rendering, Storage save, download UI
**OUT:** Batch export de múltiplas peças, agendamento de export, carrossel multi-slide

## Notas Técnicas

- Playwright precisa estar instalado no ambiente de deploy
  - Vercel: usar `@playwright/test` com Chromium bundle (heavy — considerar alternativa)
  - Alternativa: serverless function separada (AWS Lambda, Railway)
  - Alternativa dev: rodar local via CLI
- `maxDuration: 300` no route (export pode demorar)
- Considerar queue para exports em batch (futuro)
- @font-face base64 garante que fonte renderiza mesmo em headless

## File List

- `src/app/api/design/export/route.ts` — API route POST export PNG
- `src/app/clientes/[slug]/design/page.tsx` — Botões Exportar + Download
